**远程仓库**

在github上新建一个仓库，并新增ssh公钥和私钥

```
#生成公钥 gitbash内操作
ssh -keygen -t rsa 不断回车

#获取公钥
cat ~/.ssh/id_rsa.pub

#测试是否链接成功
ssh -T git@网址
ssh -T git@github.com
ssh -T git@gittee.com
```
